# 清溪镇游客故事设计规格

## 1. 概述

- **故事名称**: 初来清溪
- **类型**: 单线视觉小说风格的对话体验
- **设定**: 玩家是一名初来清溪镇的游客，刚坐班车抵达汽车站
- **目标**: 通过与本地NPC的对话，感受清溪镇的风土人情，发现松林观景台这个秘密地点

## 2. 故事大纲

### 四场景单线流程

```
[场景1] 汽车站 (车站广场)
  └─ 李阿姨主动搭话（位置触发：无条件）
  └─ 介绍清溪镇概况，推荐老街和奶茶店
  └─ 玩家可选择：接受推荐去老街 / 随便逛逛
  └─ 关系建立：认识了"本地通李阿姨"

[场景2] 老街 → 奶茶店
  └─ 玩家到达老街，小林在奶茶店主动搭话
  └─ 搭话条件：到达老街 AND 见过李阿姨
  └─ 分享高中生日常，推荐松林观景台
  └─ 关系建立：认识了"同龄朋友小林"

[场景3] 滨江步道
  └─ 玩家在河边散步，李阿姨骑自行车经过认出玩家
  └─ 搭话条件：到达滨江步道 AND 见过李阿姨
  └─ 关心游玩体验，介绍端午节龙舟活动（呼应世界观设定）
  └─ 再次推荐松林观景台

[场景4] 松林观景台 (结局)
  └─ 玩家到达山顶，夕阳时分
  └─ 小林（放学后也来放松）或李阿姨（下班后来）已在
  └─ 搭话条件：到达观景台 AND 与任一NPC关系值≥1
  └─ 温馨收尾：分享这是他们私藏的秘密地点
  └─ 结局选项：感谢李阿姨 / 感谢小林 / 感谢两人
```

## 4. 场景详细设计

### 4.1 场景1: 汽车站

**触发条件**: 玩家进入游戏，自动进入汽车站场景

**初始旁白**:
> 班车缓缓停在清溪镇汽车站。你拎着行李走下车，一股带着泥土气息的夏风迎面吹来。车站不大，水泥地面上晒着几床被子，远处停着几辆电动三轮车。一个戴着袖套的中年女性正朝你张望。

**李阿姨主动搭话内容**:
> "哎，这位是第一次来清溪镇吧？我在这儿上班快二十年了，镇上哪家店好吃、哪儿好玩，我闭着眼睛都能找到。你这是来探亲还是旅游呀？"

**玩家选项**:
- A: "来旅游，想看看你们这儿有什么好玩的地方"
- B: "随便逛逛，没什么特别计划"
- C: 自由输入

**李阿姨回应（选项A）**:
> "那可来对地方了！我们清溪镇虽然小，但有意思的地方可不少。老街那边有家奶茶店，开了十来年了，饮料好喝还能坐着歇脚。要是想看风景，出城往北走有个松林观景台，晚上看星星可美了，不过现在去正好赶上日落。"

**李阿姨回应（选项B）**:
> "随便逛逛好呀！我们这儿地方不大，随便走两步就能碰到熟人。沿着这条主街往南走是老街，街边有卖小吃的；往北是滨江步道，晚上散步的人可多了。"

**李阿姨回应（自由输入）**: 根据输入内容灵活回复，保持热情本地人角色

**场景转移选项**:
- "去老街看看" → 场景2
- "先在车站附近转转" → 可自由探索后再去老街

---

### 4.2 场景2: 老街 → 奶茶店

**触发条件**: 玩家到达老街 AND 已与李阿姨对话过

**旁白**:
> 你沿着指示牌走入老街。青石板路面被岁月打磨得光滑发亮，两侧是两层木结构瓦房，法梧树荫遮住了大半个街道。走到老街中段，一家挂着"清溪奶茶"招牌的小店出现在眼前，门口摆着几盆绿植。柜台后的男生正低头整理茶杯，似乎注意到了你的目光。

**小林主动搭话**:
> "你...你是来旅游的吗？嗯...那个，我看你不像是常来老街的人。这家店我在这儿做兼职两年了，你要是想喝点什么的话，我可以给你推荐。"

**玩家选项**:
- A: "好啊，有什么推荐的吗？"
- B: "李阿姨让我来看看的，说你们这儿不错"
- C: 自由输入

**小林回应（选项A）**:
> "真的吗！那我推荐店里的招牌柠檬茶，是我自己调的配方，酸酸甜甜的。我们这儿下午人不多，坐窗户边能看到老街的全貌，景色挺好的。对了，你接下来打算去哪儿？要是想看风景的话，可以去松林观景台，那儿是我的秘密基地。"

**小林回应（选项B）**:
> "李阿姨推荐的呀？她人可好了，在车站工作二十年，全镇的事儿她都门儿清。她说得没错，我们这儿确实不错。嗯...对了，你是来旅游的吧？松林观景台那边风景特别好，傍晚的时候去能看到日落。"

**小林回应（自由输入）**: 保持害羞但真诚的学生角色

**场景转移选项**:
- "去松林观景台" → 场景4
- "去滨江步道走走" → 场景3

---

### 4.3 场景3: 滨江步道

**触发条件**: 玩家到达滨江步道 AND 已与李阿姨对话过

**旁白**:
> 你沿着清溪河岸走在滨江步道上。夕阳把河面染成金黄色，柳枝在微风中轻轻摇曳。远处传来自行车铃声，一个穿着工作服的女性从你身边骑过，突然停了下来。

**李阿姨主动搭话**:
> "哎！是你呀！逛完老街了？怎么样，奶茶店的小林人不错吧？这一圈下来感觉我们清溪镇怎么样？"

**玩家选项**:
- A: "挺好的，比想象中热闹"
- B: "节奏好慢，好舒服"
- C: 自由输入

**李阿姨回应**:
> "是吧！我们这儿虽然比不上大城市，但该有的都有，生活起来可舒心了。每年端午节的龙舟赛可热闹了，就在前面那段河面，你要是有机会赶上就好了。对了，日落快没了，你要是还没去松林观景台，赶紧去，骑车十分钟就到，那儿看日落可美了，我每天下班有时候也去那儿坐坐。"

**场景转移选项**:
- "去松林观景台看日落" → 场景4
- "继续沿河走走" → 可自由探索后再去场景4

---

### 4.4 场景4: 松林观景台 (结局)

**触发条件**: 玩家到达松林观景台 AND 与任一NPC关系值≥1

**旁旁白**:
> 你骑车沿着蜿蜒的山路来到松林观景台。马尾松林间搭着一座木平台，视野开阔，正对西方河谷方向。夕阳正缓缓沉入天际，天边铺满了橙红色的云霞。平台上已经坐着一个人，背对着你，望着远方。

**根据关系情况决定谁先开口**:

**若与小林关系值更高**:
> 小林听到脚步声，回过头来，认出是你，脸上露出一丝惊喜："是你呀！你真的来松林观景台了...我就知道你会喜欢这里的。这个地方是我的秘密基地，平时放学后我会来这里坐坐，看看夕阳，想想事情。"

**若与李阿姨关系值更高**:
> 李阿姨转过身来，笑着招呼你："来啦！我就知道你会来这儿。这个观景台可是我们清溪镇的宝贝，本地人都知道，但一般游客很难找到。你看这日落，美不美？"

**结局选项**:
- "谢谢你推荐这个地方，李阿姨"
- "谢谢你带我来这里，小林"
- "谢谢你们两个，这里真的很棒"
- 自由输入

**结局旁白**（统一）:
> 夕阳彻底沉入河谷，天边只剩下淡淡的紫色。路灯亮了起来，山下的清溪镇渐渐亮起星星点点的灯火。你站在观景台上，听着松涛阵阵，觉得这趟清溪镇来得很值。

**THE END**

## 5. 触发系统设计

### 5.1 触发条件表

| 场景 | 触发条件 | 优先级 |
|------|----------|--------|
| 场景1-李阿姨搭话 | 无条件（游戏开始） | 0 |
| 场景2-小林搭话 | 到达老街 AND 已见过李阿姨 | 1 |
| 场景3-李阿姨搭话 | 到达滨江步道 AND 已见过李阿姨 | 1 |
| 场景4-任一NPC搭话 | 到达松林观景台 AND 关系值≥1 | 0 |

### 5.2 关系值系统

每个NPC维护一个关系值（初始0）:

| 事件 | 关系值变化 |
|------|-----------|
| 与NPC首次对话 | +1 |
| 选择接受NPC的建议 | +1 |
| NPC主动搭话后玩家积极回应 | +1 |

**关系值用途**: 场景4决定谁先开口，以及结局对话分支

## 7. 数据结构

### 7.1 故事状态

```python
class StoryState:
    current_scene: str           # "station" | "old_street" | "riverwalk" | "viewpoint"
    met_npcs: List[str]         # 已见过的NPC ID列表
    npc_relationships: Dict[str, int]  # NPC关系值
    scene_flags: Dict[str, bool]  # 场景标志位
    story_progress: int          # 故事进度 0-100
```

### 7.2 场景配置

```yaml
# assets/story/tourist_story/scenes.yaml
scenes:
  station:
    id: station
    name: 汽车站
    description: 清溪镇汽车站
    narrator_intro: "班车缓缓停在清溪镇汽车站..."
    npcs:
      - li_ayi
    triggers:
      - type: unconditional
        priority: 0
        action: npc_greet

  old_street:
    id: old_street
    name: 老街
    description: 清溪镇老街
    narrator_intro: "你沿着指示牌走入老街..."
    npcs:
      - xiao_lin
    triggers:
      - type: location_and_relationship
        location: old_street
        npc: li_ayi
        min_relationship: 1
        priority: 1
        action: npc_greet
```
