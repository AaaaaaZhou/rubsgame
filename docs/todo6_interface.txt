## 6.2 PowerShell 交互器
- [ ] 创建文件：`src/interface/ps_shell.py`
  - [ ] 定义类：`PowerShellInterface`
    - [ ] 方法：`__init__(self, engine_core: EngineCore)` (假设 EngineCore 是整合所有管理器的总控)
    - [ ] 方法：`print_status(self, session: ConversationSession)`
      - [ ] 交互：向 PowerShell 打印表格 (当前人设、记忆条数、Token 估算)
    - [ ] 方法：`run_repl(self)`
      - [ ] 逻辑：主循环
        1. 读取用户输入 `input("> ")`
        2. 检测指令：
           - `/save <name>` -> 调用 session_manager.save
           - `/load <name>` -> 调用 session_manager.load
           - `/role <name>` -> 调用 asset_manager.load_persona
           - `/emotion toggle` -> 开关情绪渲染
           - `/exit` -> 触发 memory_engine.finalize_session -> break
        3. 非指令：
           - 调用 orchestrator.build_messages
           - 调用 llm_client.chat
           - 调用 emotion_engine.process_model_output
           - 调用 session.append_message
           - 打印结果
